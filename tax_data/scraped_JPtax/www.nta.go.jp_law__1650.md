# 法令等

## 税法

電子政府の総合窓口（e-Gov）で提供している「e-Gov法令検索」で、税に関する法律、政令、省令及び規則を調べることができます。法令用語や法令名のほか、事項別の分類、法令の種類ごとに検索することができます。国税関係の法令は、「事項別分類検引」の「国税」に掲載されています。

- [税法（e-Govの「e-Gov法令検索」へリンク）](https://elaws.e-gov.go.jp/search/elawsSearch/elaws_search/lsg0100/)

## 法令解釈通達

税法についての取扱い等を定めた国税庁の通達を調べるための入り口です。各税目の基本通達、個別通達、措置法通達を調べることができるほか、キーワード検索で調べることもできます。

- キーワードで検索　→ [法令解釈通達内を検索](https://www.nta.go.jp/law/tsutatsu/menu.htm)
- 税目別の各種手続　→ [所得税](https://www.nta.go.jp/law/tsutatsu/menu.htm#a-syotoku)　/　[相続税・贈与税](https://www.nta.go.jp/law/tsutatsu/menu.htm#a-souzoku)　/　[法人税](https://www.nta.go.jp/law/tsutatsu/menu.htm#a-houzin)　/　[間接税](https://www.nta.go.jp/law/tsutatsu/menu.htm#a-kansetsu)　/　[徴収関係](https://www.nta.go.jp/law/tsutatsu/menu.htm#a-tyousyu)　/　[不服申立関係](https://www.nta.go.jp/law/tsutatsu/menu.htm#a-fufuku)　/　[その他](https://www.nta.go.jp/law/tsutatsu/menu.htm#a-sonota)
