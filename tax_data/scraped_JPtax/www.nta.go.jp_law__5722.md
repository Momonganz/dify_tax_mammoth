## 2.券面額での譲渡が行われた場合

|  | 譲渡・中間取得時の処理（発行後3か月目） | 決算時の処理（発行後4か月目） | 償還時の処理 |
| --- | :-: | :-: | :-: |
| ・　発行条件<br>　　1と同じ<br>・　経過利息　　11,250千円<br>![10億円×4.5％×3÷12](https://www.nta.go.jp/law/tsutatsu/kobetsu/hojin/790704/02.gif)<br>・　未収利息　　3,750千円<br>![10億円×4.5％×1÷12](https://www.nta.go.jp/law/tsutatsu/kobetsu/hojin/790704/03.gif) | 譲渡銀行 |  |  |  |
| 現　金<br>1,011,250 | 譲渡性預け金<br>1,000,000<br>譲渡性預け金利息<br>11,250 |
| 譲受銀行 |  | （期　末　日） |  |
| 譲渡性預け金<br>1,000,000<br>仮払金（経過利息）<br>11,250 | 現　金<br>1,011,250 | 未収利息その他<br>3,750<br>（翌期初日） | 譲渡性預け金利息<br>3,750 | 現　金<br>1,022,500 | 譲渡性預け金<br>1,000,000<br>仮払金<br>11,250<br>譲渡性預け金利息<br>11,250 |
| 未収利息その他戻入<br>3,750 | 未収利息その他<br>3,750 |

（注）発行銀行の処理は、1と同じ

## 3.券面額と異なる価額で譲渡が行われた場合

|  | 譲渡・中間取得時の処理（発行後3か月目） | 決算時の処理（発行後4か月目） | 償還時の処理 |
| --- | :-: | :-: | :-: |
| A<br>・　発行条件<br>　　　1と同じ<br>・　譲渡価額　　1,006百万円<br>・　経過利息　　11,250千円<br>・　決算時未収利息<br>3,750千円 | 譲渡銀行 |  |  |  |
| 現　金<br>1,017,250 | 譲渡性預け金<br>1,000,000<br>譲渡性預け金利息<br>17,250 |
| 譲受銀行 |  | （期　末　日） |  |
| 譲渡性預け金<br>1,000,000<br>仮払金<br>（経過利息＋券面差額）<br>17,250 | 現　金<br>1,017,250 | 未収利息<br>3,750<br>（翌期初日） | 譲渡性預け金利息<br>3,750 | 現　金<br>1,022,500 | 譲渡性預け金<br>1,000,000<br>仮払金<br>17,250<br>譲渡性預け金利息<br>5,250 |
| 未収利息その他戻入<br>（譲渡性預け金利息）<br>3,750 | 未収利息その他<br>3,750 |
| B<br>・　譲渡価額　　995百万円<br>・　その他の条件は1と同じ | 譲渡銀行 |  |  |  |
| 現　金<br>1,006,250 | 譲渡性預け金<br>1,000,000<br>譲渡性預け金利息<br>6,250 |
| 譲受銀行 |  | Aと同じ |  |
| 譲渡性預け金<br>1,000,000<br>仮払金<br>6,250 | 現　金<br>1,006,250 | 現　金<br>1,022,500 | 譲渡性預け金<br>1,000,000<br>仮払金<br>6,250<br>譲渡性預け金利息<br>16,250 |

（注）

1. 1.譲渡性預け金価額は券面額を付す。
2. 2.購入対価が券面額を超える部分は"仮払金",券面額を下廻る部分は"仮受金"で処理する。
