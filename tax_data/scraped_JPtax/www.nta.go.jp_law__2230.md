|  | 第1章　名称及び事務所 |
|  |  |
| 第 | 1条　本社団は、医療法人○○会と称する。 |
|  |  |
| 第 | 2条　本社団は、事務所を○○県○○郡（市）○○町（村）<br>○○番地に置く。 |
|  | 第2章　目的及び事業 |
|  |  |
| 第 | 3条　本社団は、病院（診療所、介護老人保健施設）を<br>経営し、科学的でかつ適正な医療（及び疾病・負傷等<br>により寝たきりの状態等にある老人に対し、看護、医<br>学的管理下の介護及び必要な医療等）を普及すること<br>を目的とする。 |
|  |  |
|  |  |
| 第 | 4条　本社団の開設する病院（診療所、介護老人保健施

設）の名称及び開設場所は、次のとおりとする。

| (3) | ○○園 | ○○県○○郡（市） | ○○町（村） | |
|  |  |
| 第 | 5条　本社団は、前条に掲げる病院（診療所、介護老人<br>保健施設）を経営するほか、次の業務を行う。<br>○○看護師養成所の経営 |
|  |  |
|  | 第3章　社員 |
|  |  |
| 第 | 6条　本社団の社員になろうとするものは、社員総会の<br>承認を得なければならない。 |
|  |  |
| 第 | 7条　社員は、次に掲げる理由によりその資格を失う。

|     |     |
| --- | --- |
| 1 | 除名 |
| 2 | 死亡 |
| 3 | 退社 | |
| 2 | 社員であって、社員たる義務を履行せず本社団の定款<br>に違反し又は品位を傷つける行為のあつた者は、社員総<br>会の議決を経て除名することができる。 |
|  |  |
| 第 | 8条　やむを得ない理由のあるときは、社員はその旨を<br>理事長に届け出て、その同意を得て退社することができ<br>る。 |
|  |  |
| 第 | 9条　社員資格を喪失した者は、その出資額を限度とし<br>て払戻しを請求することができる。 |
|  |  |
|  | 第4章　資産及び会計 |
|  |  |
| 第 | 10条　本社団の資産のうち、次に掲げる財産を基本財

産とする。

|     |     |
| --- | --- |
| (1) | ・・・ |
| (2) | ・・・ |
| (3) | ・・・ | |
| 2 | 基本財産は処分し、又は担保に供してはならない。た<br>だし、特別の理由のある場合には、理事会及び社員総会<br>の議決を経て、処分し、又は担保に供することができる。 |
|  |  |
| 第 | 11条　本社団の資産は、社員総会で定めた方法によっ<br>て、理事長が管理する。 |
|  |  |
| 第 | 12条　資産のうち現金は、日本郵政公社、確実な銀行<br>又は信託会社に預け入れ若しくは信託し、又は国公債若<br>しくは確実な有価証券に換え保管するものとする。 |
|  |  |
| 第 | 13条　本社団の収支予算は、毎会計年度開始前に理事<br>会及び社員総会の議決を経て定める。 |
|  |  |
| 第 | 14条　本社団の会計年度は、毎年4月1日に始まり翌<br>年3月31日に終る。 |
|  |  |
| 第 | 15条　本社団の決算については、毎会計年度終了後2<br>月以内に監事の監査を経た上、理事会及び社員総会の承<br>認を受け、かつ、これを○○県知事（○○厚生局長）に<br>届け出なければならない。 |
|  |  |
| 第 | 16条　決算の結果、剰余金を生じたときは、理事会及<br>び社員総会の議決を経てその全部又は一部を基本財産<br>に繰り入れ、又は積立金として積み立てるものとし、配<br>当してはならない。 |

|  | 第5章　役員 |
|  |  |
| 第 | 17条　本社団に、次の役員を置く。

|     |     |
| --- | --- |
| (1) | |     |     |
| --- | --- |
| 理事 | ○名以上○名以内 |
| うち | 理事長1名 |
|  | 常務理事○名 | |
| (2) | 監事　○名 | |
|  |  |
| 2 | 理事及び監事は、社員総会において本社団の社員の中<br>から選任する。ただし、必要があるときは、社員以外の者<br>から選任することを妨げない。 |
|  |  |
| 第 | 18条　理事長及び常務理事は、理事の互選によって定<br>める。 |
| 2 | 本社団の開設する病院（診療所、介護老人保健施設）<br>の管理者は、必ず理事に加えなければならない。ただし、<br>○○県知事（○○厚生局長）の認可を受けた場合はこの<br>限りでない。 |
| 3 | 前項の理事は、管理者の職を退いたときは、理事の職<br>を失うものとする。ただし、再選を妨げるものではない。 |
|  |  |
| 第 | 19条　理事長のみが本社団を代表する。 |
| 2 | 理事長は本社団の業務を総理する。 |
| 3 | 常務理事は、理事長を補佐して常務を処理し、理事長<br>に事故があるときは、その職務を行う。 |
| 4 | 理事は、本社団の常務を処理する。 |
| 5 | 監事は、民法第59条に規定する職務を行う。 |
| 6 | 監事は、この法人の理事又は他の職務を兼任すること<br>ができない。 |
|  |  |
| 第 | 20条　役員の任期は2年とする。ただし、再任を妨げ<br>ない。 |
| 2 | 補欠により就任した役員の任期は、前任者の残留期間<br>とする。 |
| 3 | 役員は、任期満了後といえども、後任者の就任するま<br>では、その職務を行うものとする。 |
|  |  |
|  | 第6章　会議 |
|  |  |
| 第 | 21条　会議は、社員総会及び理事会の2つとし、社員<br>総会はこれを定時総会と臨時総会に分ける。 |
|  |  |
| 第 | 22条　定時総会は、毎年2回、○月及び○月に開催し、<br>臨時総会及び理事会は随時必要なときに開催する。 |
|  |  |
| 第 | 23条　会議は、理事長がこれを招集し、その議長とな<br>る。 |
| 2 | その会議を構成する社員又は理事の3分の1以上か<br>ら連名をもって会議の目的たる事項を示して請求があ<br>ったときは、理事長はその会議を招集しなければならな<br>い。 |
|  |  |
| 第 | 24条　社員総会は、社員の2分の1以上が出席しなけ<br>れば、議事を開くことができない。 |
|  |  |
| 第 | 25条　次の事項は、社員総会の議決を経なければなら

ない。

|     |     |
| --- | --- |
| 1 | 定款の変更 |
| 2 | 基本財産の設定及び処分（担保提供を含む。) |
| 3 | 毎事業年度の事業計画の決定及び変更 |
| 4 | 収支予算及び決算の決定 |
| 5 | 剰余金又は損失金の処理 |
| 6 | 借入金額の最高限度の決定 |
| 7 | 社員の入社及び除名 |
| 8 | 本社団の解散 |
| 9 | 他の医療法人との合併契約の締結 |
| 10 | その他重要な事項 | |
|  |  |
| 第 | 26条　社員総会の議事は、別段の定めあるもののほか、<br>出席した社員の議決権の過半数で決し、可否同数のとき<br>は、議長の決するところによる。ただし、定款の変更、<br>社員の除名及び解散の議決は、社員の3分の2以上が出<br>席し、その3分の2以上の同意を要する。 |
|  |  |
| 第 | 27条　社員総会の招集は、期日の少なくとも5日前ま<br>でに会議の目的である事項、日時及び場所を記載し、理<br>事長がこれに記名した書面で社員に通知しなければな<br>らない。 |
| 2 | 社員総会においては、前項の規定によってあらかじめ<br>通知した事項のほか議決することができない。ただし、<br>急を要する場合はこの限りではない。 |
|  |  |
| 第 | 28条　社員は、社員総会において1個の議決権及び選<br>挙権を有する。 |
|  |  |
| 第 | 29条　社員は、あらかじめ通知のあった事項について<br>のみ書面又は代理人をもって議決権及び選挙権を行使<br>することができる。ただし、代理人は社員でなければな<br>らない。 |
| 2 | 代理人は、代理権を証する書面を議長に提出しなけれ<br>ばならない。 |
|  |  |
| 第 | 30条　会議の議決事項につき特別の利害関係を有する<br>者は、当該事項につきその議決権を行使できない。 |
|  |  |
| 第 | 31条　社員総会の議事についての細則は、社員総会で<br>定める。 |
| 2 | 理事会の議事についての細則は、理事会で定める。 |

|  | 第7章　定款の変更及び解散 |
|  |  |
| 第 | 32条　この定款は、社員総会の議決を経、かつ、○○<br>県知事(○○厚生局長)の認可を得なければ変更すること<br>ができない。 |
|  |  |
| 第 | 33条　本社団が解散したときは、理事がその清算人と<br>なる。ただし、社員総会の議決によって社員の中からこ<br>れを選任することができる。 |
|  |  |
| 第 | 34条　本社団が解散した場合の残余財産は、払込済出<br>資額を限度として分配するものとし、当該払込済出資額<br>を控除してなお残余があるときは、社員総会の議決によ<br>り、○○県知事（厚生労働大臣）の認可を得て、国若し<br>くは地方公共団体又は租税特別措置法（昭和32年法律<br>第26号）第67条の2に定める特定医療法人若しくは<br>医療法（昭和23年法律第205号）第42条第2項に定<br>める特別医療法人に当該残余の額を帰属させるものと<br>する |
|  |  |
| 第 | 34条の2　第9条及び前条の規定は第32条の規定にか<br>かわらず変更することができない。ただし、特定医療法<br>人又は特別医療法人に移行するために変更する場合は<br>この限りではない。 |
