# 「平成18年分の基準年利率について」の一部改正について(法令解釈通達)

平成18年5月18日付課評2－9「平成18年分の基準年利率について」（法令解釈通達）について、4月分から6月分の基準年利率を定め、下記のとおり改正したから、これによられたい。

記

○　基準年利率

（単位：％）

| 区分 | 年数<br>又は<br>期間 | 平成<br>18年<br>1月 | 2月 | 3月 | 4月 | 5月 | 6月 | 7月 | 8月 | 9月 | 10月 | 11月 | 12月 |
| --- | :-: | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 短期 | 1年 | 0.25 | 0.25 | 0.25 | 0.5 | 0.5 | 0.75 |  |  |  |  |  |  |
| 2年 |
| 中期 | 3年 | 0.75 | 0.75 | 1.0 | 1.5 | 1.0 | 1.5 |  |  |  |  |  |  |
| 4年 |
| 5年 |
| 6年 |
| 長<br>期 | 7年<br>以上 | 1.5 | 1.5 | 1.5 | 2.0 | 2.0 | 2.0 |  |  |  |  |  |  |

（注）　課税時期の属する月の年数又は期間に応ずる基準年利率を用いることに留意する。
