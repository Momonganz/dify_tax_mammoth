# 長期間保有していた土地に区画形質の変更を加えて譲渡した場合の所得計算

## 【照会要旨】

甲は、20年前に取得した土地5,000![平方メートル](https://www.nta.go.jp/shared/images/mark/m2.gif)に区画形質の変更を加え、分譲地として売却しました。次の例における所得金額の計算は次の2案のうちいずれによるべきでしょうか。

(例)

![1](https://www.nta.go.jp/shared/images/decimal/01.gif)　土地の取得価額 1,200万円

![2](https://www.nta.go.jp/shared/images/decimal/02.gif)　区画形質変更直前の価額 10,000万円

![3](https://www.nta.go.jp/shared/images/decimal/03.gif)　造成費用・販売経費 2,000万円

![4](https://www.nta.go.jp/shared/images/decimal/04.gif)　販売価額 20,000万円

![5](https://www.nta.go.jp/shared/images/decimal/05.gif)　区画形質変更後に寄附した道路用地 500![平方メートル](https://www.nta.go.jp/shared/images/mark/m2.gif)

![所得金額の計算例の図](https://www.nta.go.jp/law/shitsugi/joto/02/img/02_01.gif)

|     |     |     |
| --- | --- | --- |
| (2案) | 譲渡所得 | 10,000万円－1,200万円=8,800万円 |
|  | 雑所得 | 20,000万円－10,000万円－2,000万円=8,000万円 |

## 【回答要旨】

2案によります。

## 【関係法令通達】

所得税基本通達33-5
